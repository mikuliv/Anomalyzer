# Anomalyzer / Filin

Репозиторий содержит историческую версию проекта Anomalyzer и новую ветку развития платформы Filin.

## Структура

- `Anomalyzer-main/` - базовая версия проекта Anomalyzer, выполненная как выпускная квалификационная работа СПО по специальности 10.02.05 «Обеспечение информационной безопасности автоматизированных систем».
- `filin/` - новая версия платформы для развития проекта в рамках ВУЗа.

## Цель развития

Цель развития проекта - переход от простого ML-классификатора сетевых аномалий к платформе мониторинга, анализа и поддержки расследования инцидентов информационной безопасности.

Новая платформа должна объединить сбор сетевых событий, извлечение признаков, ML-детекцию, формирование карточек инцидентов, сопоставление с MITRE ATT&CK, подготовку Sigma-кандидатов и проверку правил на лабораторном стенде.
